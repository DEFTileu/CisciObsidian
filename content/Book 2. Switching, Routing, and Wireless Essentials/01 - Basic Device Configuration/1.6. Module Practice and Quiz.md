## 1.6.3 Final understanding of Module 1
Если прочитал и понял весь модуль, у тебя в голове должна сложиться такая картина:
### 1. Switch startup
Switch проходит путь:
`POST -> boot loader -> flash -> IOS`
### 2. Switch management
Для remote management нужен:
- `SVI`;
- management IP;
- иногда `default gateway`.
### 3. Switch ports
Ты должен понимать:
- `half-duplex`;
- `full-duplex`;
- `speed`;
- `auto-MDIX`;
- counters ошибок.
### 4. Secure remote access
Ты должен чётко различать:
- `Telnet` = insecure;
- `SSH` = secure.
### 5. Router basics
Ты должен уметь:
- сделать initial router setup;
- настроить interfaces;
- включить их.
### 6. Verification
Ты должен читать:
- interface status;
- routing table;
- `ping`;
- filtered output.
> [!abstract] Самое важное для запоминания
> - `Telnet = TCP 23`
> - `SSH = TCP 22`
> - `FE80 = link-local`
> - `C = Connected`
> - `L = Local`
> - `administratively down` -> `no shutdown`
> - `SVI` = switch management
## 1.6.4 Quiz mindset
На quiz лучше идти не с набором разрозненных команд, а с пониманием связей:
- зачем команда вводится;
- в каком режиме она работает;
- что ты потом проверяешь после неё.
> [!success] Главная идея модуля
> `Configure -> Secure -> Verify`
