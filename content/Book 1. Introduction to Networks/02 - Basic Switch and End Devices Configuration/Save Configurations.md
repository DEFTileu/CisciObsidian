## 2.5.1 Файлы конфигурации

![[Screenshot 2026-04-13 at 15.11.21.png|434]]

Два файла, где хранится конфигурация устройства:

|Файл|Где хранится|Что содержит|Пропадает при перезагрузке?|
|---|---|---|---|
|**running-config**|RAM|Текущая активная конфигурация|✅ Да|
|**startup-config**|NVRAM|Конфигурация, загружаемая при старте|❌ Нет|

**Ключевые команды:**

```
show running-config       ← посмотреть текущую конфигурацию (RAM)
show startup-config       ← посмотреть сохранённую конфигурацию (NVRAM)
copy running-config startup-config   ← СОХРАНИТЬ изменения
```

> **Важно:** если ты настроил устройство, но не выполнил `copy running-config startup-config` — после перезагрузки всё потеряется.

## 2.5.2 Изменение / сброс конфигурации

**Если сделал ошибку и ещё не сохранял:**

```
reload      ← перезагрузить устройство
            ← IOS спросит: сохранить? → введи n или no
```

Устройство загрузится с последней сохранённой конфигурацией. Минус — кратковременный простой сети.

**Если ошибочные настройки уже сохранены в startup-config:**

```
erase startup-config    ← стереть NVRAM
reload                  ← перезагрузить
```

Устройство вернётся к заводским настройкам.

## 2.5.4 Сохранение конфигурации в текстовый файл (через PuTTY)

Полезно для резервной копии или документации. Делается в 4 шага:

**Шаг 1.** Открыть PuTTY и подключиться к устройству

![[Screenshot 2026-04-13 at 15.11.58.png|499]]

**Шаг 2.** В PuTTY: `Session → Logging` → выбрать **All session output** → указать имя файла (например, `MySwitchLogs`)

![[Screenshot 2026-04-13 at 15.12.23.png|488]]

**Шаг 3.** Выполнить команду:

```
show running-config
```

Весь вывод автоматически запишется в файл.

**Шаг 4.** В PuTTY снова зайти в `Logging` → выбрать **None** (отключить логирование)

![[Screenshot 2026-04-13 at 15.13.09.png|498]]

---

**Восстановление конфигурации из текстового файла:**

1. Войти в `configure terminal`
2. Скопировать содержимое файла и вставить прямо в терминал
3. Все строки выполнятся как команды и станут текущей конфигурацией

> **Аналогия:** `running-config` — это как несохранённый документ в Word. `startup-config` — сохранённый файл. `copy running startup` — нажать Ctrl+S.
